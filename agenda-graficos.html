Agenda de Gráficos

Visualize e gerencie a agenda de tratamentos com gráficos radiestésicos. Acompanhe os agendamentos diretamente no Google Calendar integrado abaixo, organizando as sessões de tratamento e os gráficos radiestésicos de cada cliente.

Instruções de Uso

  • Utilize o calendário acima para visualizar todos os agendamentos de tratamentos com gráficos radiestésicos.
  • Clique sobre um evento no calendário para visualizar os detalhes do tratamento agendado.
  • Para criar um novo agendamento, acesse diretamente o Google Calendar vinculado a este sistema.
  • Os horários exibidos estão no fuso horário de São Paulo (America/Sao_Paulo).
  • Mantenha os eventos atualizados para garantir o acompanhamento correto dos gráficos radiestésicos de cada cliente.
  • Em caso de dúvidas sobre um tratamento específico, consulte a Lista de Tratamentos no menu lateral.
